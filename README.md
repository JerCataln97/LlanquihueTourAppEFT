📘 Descripción general del sistema

Este proyecto corresponde a la evaluación final transversal de la asignatura Desarrollo Orientado a Objetos I.

El proyecto fue creado a partir de una problemática en una agencia de turismo, en este se aplico el uso de encapsulamiento, constructores, getters y setters, metodo toString, herencia, polimorfismo, interface y colecciones.

🧱 Estructura general del proyecto

📁 src/

├── app/ # Clase principal con el método Main

├── data/ # Clases para gestionar los datos (GestorEntidades, GestorExperiencias)

├── interfaces/ # Interface (Registrable)

├── model/ # Superclase y Subclases (Persona, Empleado, Proveedor, Cliente, Experiencia)

└── utils/ # Archivos .txt (datos.txt, experiencias.txt)
